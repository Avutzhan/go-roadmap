# 1) Converting an integer to an 8-bit binary representation and back

## База: как число превращается в бинарку

👉 Алгоритм:

* Делим число на 2
* Запоминаем остаток (0 или 1)
* Повторяем, пока число не станет 0
* Собираем остатки снизу вверх (в обратном порядке)

Пример: 13 → бинарка

Делаем по шагам:

```aiignore
13 / 2 = 6, остаток 1
6  / 2 = 3, остаток 0
3  / 2 = 1, остаток 1
1  / 2 = 0, остаток 1
```
Теперь собираем снизу вверх:

`1 1 0 1`

Ответ: 1101

## Сделаем до 8 бит (как ты хотел)

Просто добавляем нули слева:

`1101 → 00001101`

👉 Это и есть 8-bit representation

## База: как бинарка превращается в число

Берём бинарное число:

`1101`

Каждая позиция это степень двойки

```aiignore
позиции:   3   2   1   0
биты:      1   1   0   1
```
Теперь считаем:
```aiignore
1 * 2³ = 8
1 * 2² = 4
0 * 2¹ = 0
1 * 2⁰ = 1
```
Складываем:

`8 + 4 + 0 + 1 = 13`

Коротко все что мы сделали можем показать примерно так

`13 = 2³ + 2² + 2⁰`

Есть шорткат как это делать нам нужно выучить степени двойки
тогда будешь решать такие вещи в уме

```aiignore
2⁰ = 1
2¹ = 2
2² = 4
2³ = 8
2⁴ = 16
2⁵ = 32
2⁶ = 64
2⁷ = 128
2⁸ = 256
```
но тут еще один трюк чтобы даже не учить это мы можем запомнить что каждая следующая степень = умножить на 2

`1 → 2 → 4 → 8 → 16 → 32 → 64 → 128`
